# ParaGames (BETA) has been UPDATED!
What ParaGames has added.

1. Leaderboard and Friends: You can now view your friends' play history!
2. Messages: You can now chat with other friends!
3. Admin Panels: Owners and admins can ban, delete, promote, and do much more to other users.
4. Forgot your password: Users can use secret codes. If you forgot your password, if you have a secret code, you can answer the secret question and make a new password!
